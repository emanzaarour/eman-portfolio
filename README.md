# Eman Zaarour — Premium Portfolio Website

This is a complete, responsive portfolio website created for Eman Zaarour based on the provided CV.

## Files included

- `index.html` — all website sections and content
- `style.css` — premium responsive design, dark/light theme, animations
- `script.js` — interactions: menu, theme switch, filters, modals, counters, contact form
- `assets/` — SVG visuals, favicon, and CV PDF

## How to open

1. Unzip the folder.
2. Open `index.html` in Chrome, Safari, Edge, or Firefox.

## How to publish

You can upload the folder to GitHub Pages, Netlify, Vercel, or any hosting service. Keep the folder structure exactly the same so the CSS, JavaScript, images, and CV link work correctly.

## How to customize

- Edit text directly in `index.html`.
- Change colors in the `:root` section of `style.css`.
- Replace `assets/Eman_Zaarour_CV.pdf` with an updated CV using the same file name.
- Replace SVG images in `assets/` if you want different visuals.
